# Curso: Programación ABAP Básica (S4D400)

[cite_start]Repositorio dedicado a los ejercicios y proyectos desarrollados durante la formación **S4D400 - Programación ABAP básica** (Versión del curso: 24).

## Descripción
[cite_start]Este curso proporciona una introducción universal al desarrollo ABAP, abarcando desde conceptos básicos de sintaxis hasta el modelo de programación de aplicaciones ABAP RESTful (RAP).

## Estructura del Repositorio
[cite_start]El contenido sigue la organización de los capítulos del manual[cite: 5, 6]:

- [cite_start]`Capítulo 1: Introducción`: Configuración del entorno (Eclipse/ADT), creación de proyectos ABAP Cloud y paquetes[cite: 5, 8].
- [cite_start]`Capítulo 2: Técnicas básicas`: Declaración de datos, estructuras de control, tablas internas y debugging[cite: 5, 7, 9].
- [cite_start]`Capítulo 3: Clases Locales`: Definición, instanciación y uso de métodos y encapsulación[cite: 5].
- [cite_start]`Capítulo 4: Base de Datos`: Sentencias SELECT básicas y uso de vistas CDS[cite: 5, 17].
- [cite_start]`Capítulo 5-6: Objetos complejos`: Trabajar con estructuras y tablas internas complejas[cite: 6].
- [cite_start]`Capítulo 7-8: Modelo RAP`: Implementación de Business Objects y servicios de IU OData[cite: 6].

## Herramientas Necesarias
Para ejecutar los ejercicios, asegúrate de tener:
1. [cite_start]**Eclipse IDE** (descargable en [eclipse.org](https://www.eclipse.org))[cite: 3].
2. [cite_start]**ABAP Development Tools (ADT)** instaladas en Eclipse mediante la URL: `tools.hana.ondemand.com/last`[cite: 3].

## Convenciones de Código
- [cite_start]Se recomienda el uso de **Finalización automática** (`Ctrl + Espacio`) para nombres de elementos y parámetros[cite: 20].
- Accesos rápidos útiles:
    - [cite_start]`F1`: Ayuda de idioma ABAP[cite: 6].
    - [cite_start]`F2`: Información sobre el elemento[cite: 6].
    - [cite_start]`F3`: Navegar a la definición[cite: 6].
    - [cite_start]`Ctrl + F3`: Activar objeto[cite: 10].
    - [cite_start]`F9`: Ejecutar como aplicación de consola[cite: 10].
